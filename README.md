# MP42PNG

> MP4 TO PNG.ZIP

---

### 1. 环境准备
```bash
# 创建并激活虚拟环境
uv venv .venv
source .venv/bin/activate # macOS/Linux
.venv\Scripts\activate    # Windows

# 安装依赖
uv pip install -r requirements.txt
```

### 2. 准备视频
```bash
mkdir video
```
将你的 `.mp4` 文件放入 `video` 文件夹中。

### 3. 开始转换
```bash
python main.py
```
